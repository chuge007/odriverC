# Qt程序发送指令操作指南

## 在自定义发送面板中的正确输入方法

### 界面说明

你的Qt程序应该有一个"自定义发送"面板，包含以下输入框：
- **Address** (地址)
- **DLC** (数据长度)
- **Payload** (数据内容)

---

## 示例1：让节点1进入闭环控制

### 输入内容：

```
Address:  0x027
DLC:      4
Payload:  08 00 00 00
```

### 详细说明：

1. **Address输入框**：输入 `0x027` 或 `39`
   - 可以输入十六进制 `0x027`
   - 也可以输入十进制 `39`
   - 两者是等价的

2. **DLC输入框**：输入 `4`
   - 表示发送4个字节的数据

3. **Payload输入框**：输入 `08 00 00`
   - 每个字节用空格分开
   - `08 00 00 00` 表示状态8（闭环控制）
   - 这是小端格式的32位整数

## 示例7：节点1位置控制（完整流程）

### 步骤1：设置控制模式为位置控制

```
Address:  0x02B
DLC:      8
Payload:  03 00 00 01 00 00
```

**Payload解释**：
- `03 00 00` = 控制模式3（位置控制）
- `01 00 00` = 输入模式1（Passthrough）

### 步骤2：发送位置命令（转1圈）

```
Address:  0x02C
DLC:      8
Payload:  00 80 3F 00 00 00
```

**Payload解释**：
- `00 00 80 3F` = 1.0（float，小端格式）= 位置1圈
- `00 00 00` = 0.0（float）= 速度前馈0

## 常用Axis State值（用于Set_Axis_State命令）

| 状态名称 | 值 | Payload |
|------|---|---------|
| IDLE（空闲） | 1 | `01 00 00 00` |
| MOTOR_CALIBRATION（电机校准） | 3 | `03 00 00` |
| FULL_CALIBRATION_SEQUENCE（完整校准） | 4 | `04 00 00 00` |
| ENCODER_OFFSET_CALIBRATION（编码器校准） | 6 | `06 00 00` |
| CLOSED_LOOP_CONTROL（闭环控制） | 8 | `08 00 00` |

## 注意事项

1. **Address格式**：
   - 可以输入十六进制（如 `0x027`）
   - 也可以输入十进制（如 `39`）
   - 两者等价，选择你习惯的即可

2. **Payload格式**：
   - 字节之间用空格分开
   - 使用十六进制表示（如 `08 00 00`）
   - 注意小端字节序（低字节在前）

3. **DLC值**：
   - 必须与Payload的字节数匹配
   - 例如：`08 00 00 00` 是4个字节，DLC就是4

4. **发送顺序**：
   - 先清除错误（如果有）
   - 再发送校准或控制命令
   - 等待命令执行完成

## 错误示例（不要这样做）

❌ **错误1**：使用不存在的节点
```
Address:  0x0CF  ← 这是节点6，不存在！
DLC:      8
Payload:  07 00 00 00 00
```

❌ **错误2**：DLC与Payload不匹配
```
Address:  0x027
DLC:      8  ← 错误！Payload只有4字节
Payload:  08 00 00 00
```

❌ **错误3**：Payload格式错误
```
Address:  0x027
DLC:      4
Payload:  8← 错误！应该是 08 00 00 00
```

## 验证方法

发送命令后，在接收窗口应该能看到：
- ODrive的响应消息
- 状态更新
- 如果没有响应，检查Address是否正确

---

## 推荐测试流程

1. **清除错误**：
   ```
   Address: 0x038 (节点1) 或 0x218 (节点16)
   DLC: 1
   Payload: 00
   ```

2. **进入闭环**：
   ```
   Address: 0x027 (节点1) 或 0x207 (节点16)
   DLC: 4
   Payload: 08 00 00
   ```

3. **设置控制模式**：
   ```
   Address: 0x02B (节点1) 或 0x20B (节点16)
   DLC: 8
   Payload: 03 00 00 01 00 00 00
   ```

4. **发送位置命令**：
   ```
   Address: 0x02C (节点1) 或 0x20C (节点16)
   DLC: 8
   Payload: 00 00 80 3F 00 00 00
   ```

按照这个顺序操作，电机应该会响应并转动！

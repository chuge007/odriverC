# 高级参数面板实现方案

## 📋 需求概述

根据官方ODrive文档，在高级页面添加完整的监控和控制参数显示，分为两大类：

### ✅ 监控参数（Telemetry / Feedback）- 实时数据
### ✅ 控制参数（Control / Config）- 可配置参数

---

## 🎯 核心参数清单

### 🥇 必看监控参数（5个）
```cpp
vbus_voltage          // 电源电压
pos_estimate          // 位置估算
vel_estimate          // 速度估算
Iq_setpoint          // 目标电流
Iq_measured          // 实际电流
```

### 🥇 必调控制参数（5个）
```cpp
pos_gain             // 位置增益
vel_gain             // 速度增益
vel_integrator_gain  // 速度积分增益
current_lim          // 电流限制
vel_limit            // 速度限制
```

### 🥈 扩展监控参数
```cpp

encoder.count_in_cpr

// 电机状态

motor.foc.Id_measured
```

### 阶段2：创建UI布局（mainwindow.cpp）

在 `buildAdvancedContent()` 中添加：

```cpp
void MainWindow::buildAdvancedParametersPanel(QVBoxLayout *mainLayout, QWidget *parent)
{
    // 创建滚动区域（因为参数很多）
    QScrollArea *paramsScrollArea = new QScrollArea(parent);
    QWidget *paramsWidget = new QWidget;
    QVBoxLayout *paramsLayout = new QVBoxLayout(paramsWidget);
    
    // 1. 核心监控参数组
    QGroupBox *coreMonitorGroup = new QGroupBox("核心监控参数（实时）", paramsWidget);
    QGridLayout *coreMonitorLayout = new QGridLayout(coreMonitorGroup);
    
    coreMonitorLayout->addWidget(new QLabel("电源电压:"), 0, 0);
    m_vbusVoltageLabel = createValueLabel();
    coreMonitorLayout->addWidget(m_vbusVoltageLabel, 0, 1);
    coreMonitorLayout->addWidget(new QLabel("V"), 0, 2);
    
    coreMonitorLayout->addWidget(new QLabel("位置估算:"), 1, 0);
    m_posEstimateLabel = createValueLabel();
    coreMonitorLayout->addWidget(m_posEstimateLabel, 1, 1);
    coreMonitorLayout->addWidget(new QLabel("圈"), 1, 2);
    
    coreMonitorLayout->addWidget(new QLabel("速度估算:"), 2, 0);
    m_velEstimateLabel = createValueLabel();
    coreMonitorLayout->addWidget(m_velEstimateLabel, 2, 1);
    coreMonitorLayout->addWidget(new QLabel("圈/秒"), 2, 2);
    
    coreMonitorLayout->addWidget(new QLabel("Iq设定:"), 3, 0);
    m_iqSetpointLabel = createValueLabel();
    coreMonitorLayout->addWidget(m_iqSetpointLabel, 3, 1);
    coreMonitorLayout->addWidget(new QLabel("A"), 3, 2);
    
    coreMonitorLayout->addWidget(new QLabel("Iq测量:"), 4, 0);
    m_iqMeasuredLabel = createValueLabel();
    coreMonitorLayout->addWidget(m_iqMeasuredLabel, 4, 1);
    coreMonitorLayout->addWidget(new QLabel("A"), 4, 2);
    
    paramsLayout->addWidget(coreMonitorGroup);
    
    // 2. PID参数组
    QGroupBox *pidGroup = new QGroupBox("PID参数配置", paramsWidget);
    QGridLayout *pidLayout = new QGridLayout(pidGroup);
    
    pidLayout->addWidget(new QLabel("位置增益 (pos_gain):"), 0, 0);
    m_posGainSpin = new QDoubleSpinBox(pidGroup);
    m_posGainSpin->setRange(0.0, 1000.0);
    m_posGainSpin->setDecimals(3);
    m_posGainSpin->setValue(8.0);
    pidLayout->addWidget(m_posGainSpin, 0, 1);
    
    pidLayout->addWidget(new QLabel("速度增益 (vel_gain):"), 1, 0);
    m_velGainSpin = new QDoubleSpinBox(pidGroup);
    m_velGainSpin->setRange(0.0, 10.0);
    m_velGainSpin->setDecimals(6);
    m_velGainSpin->setValue(0.05);
    pidLayout->addWidget(m_velGainSpin, 1, 1);
    
    pidLayout->addWidget(new QLabel("速度积分增益 (vel_integrator_gain):"), 2, 0);
    m_velIntegratorGainSpin = new QDoubleSpinBox(pidGroup);
    m_velIntegratorGainSpin->setRange(0.0, 10.0);
    m_velIntegratorGainSpin->setDecimals(6);
    m_velIntegratorGainSpin->setValue(0.2);
    pidLayout->addWidget(m_velIntegratorGainSpin, 2, 1);
    
    m_applyPidButton = new QPushButton("应用PID参数", pidGroup);
    pidLayout->addWidget(m_applyPidButton, 3, 0, 1, 2);
    
    paramsLayout->addWidget(pidGroup);
    
    // ... 继续添加其他参数组
    
    paramsScrollArea->setWidget(paramsWidget);
    paramsScrollArea->setWidgetResizable(true);
    mainLayout->addWidget(paramsScrollArea);
}
```

### 阶段3：实现数据读取

```cpp
void MainWindow::refreshAdvancedParameters()
{
    ODriveMotorController *controller = currentDebugController();
    if (!controller || !controller->isConnected()) {
        return;
    }
    
    const quint8 nodeId = currentDebugNodeId();
    const ODriveMotorController::AxisStatus &status = controller->status(nodeId);
    
    // 更新监控参数
    m_vbusVoltageLabel->setText(QString::number(status.busVoltageVolts, 'f', 2));
    m_posEstimateLabel->setText(QString::number(status.positionTurns, 'f', 3));
    m_velEstimateLabel->setText(QString::number(status.velocityTurnsPerSecond, 'f', 3));
    m_iqSetpointLabel->setText(QString::number(status.iqSetpointAmps, 'f', 3));
    m_iqMeasuredLabel->setText(QString::number(status.iqMeasuredAmps, 'f', 3));
}
```

### 阶段4：实现参数应用

```cpp
void MainWindow::applyPidParameters()
{
    ODriveMotorController *controller = currentDebugController();
    if (!controller) {
        return;
    }
    
    const quint8 nodeId = currentDebugNodeId();
    
    // 发送PID参数到ODrive
    // 注意：需要在 ODriveMotorController 中添加相应的方法
    controller->setPidGains(nodeId,
                           static_cast<float>(m_posGainSpin->value()),
                           static_cast<float>(m_velGainSpin->value()),
                           static_cast<float>(m_velIntegratorGainSpin->value()));
    
    appendLog(QString("已应用PID参数: pos_gain=%1, vel_gain=%2, vel_integrator_gain=%3")
              .arg(m_posGainSpin->value())
              .arg(m_velGainSpin->value())
              .arg(m_velIntegratorGainSpin->value()));
}
```

{

    // 新增：设置编码器参数

    // 新增：设置电机参数

    // 新增：设置轨迹参数

    // 新增：读取配置参数

};

## ⚠️ 注意事项

1. **实时更新**：监控参数需要在 `updateStatusPanel()` 中更新
2. **参数验证**：应用参数前需要验证范围
3. **错误处理**：参数应用失败需要提示用户
4. **保存配置**：修改的参数需要调用 `save_configuration()`
5. **权限控制**：某些参数只能在特定状态下修改

## 🔴 发现的关键问题

### 1. **轴错误 0x00004000** ⚠️⚠️⚠️

```
轴错误 (axis.error): 0x00004000
```

**这是震动的根本原因！**

#### 错误代码解析
`0x00004000` = 十进制 16384 = `AXIS_ERROR_CONTROLLER_FAILED`

**含义**：控制器失败错误

**原因**：
- 控制器无法正常工作
- 可能是控制器超时或控制死区问题
- 电机在尝试控制时失败

#### 为什么会导致震动？
当轴有 `CONTROLLER_FAILED` 错误时：
1. 电机会尝试进入闭环控制
2. 控制器检测到异常并失败
3. 电机可能在"尝试控制→失败→重试"之间循环
4. 导致电机抖动/震动

### 2. **Iq 电流跟踪误差过大** ⚠️

```
Iq设定: -10.000 A
Iq测量: -7.647 A
Iq跟踪误差: 2.353 A
```

**问题分析**：
- 设定电流为 -10A（负值表示反向）
- 实际测量只有 -7.647A
- 误差 2.353A 超过了 2A 的阈值

**可能原因**：
1. **电流限制不足**：当前限制为 10A，可能不够
2. **电源电压不足**：无法提供足够的电流
3. **电机负载过大**：需要更大的电流才能驱动
4. **PWM 饱和**：电压不足导致无法达到目标电流

## 🔍 震动的根本原因总结

### 主要原因：AXIS_ERROR_CONTROLLER_FAILED (0x00004000)

**完整的故障链**：

1. **用户发送运动命令**（速度或位置）
2. **电机尝试进入闭环控制**
3. **控制器检测到问题**：
   - 可能是电流跟踪误差过大（2.353A）
   - 可能是控制超时
   - 可能是电源电压不足
4. **控制器失败并报错** → `AXIS_ERROR_CONTROLLER_FAILED`
5. **电机被强制退出到空闲状态**
6. **如果用户继续发送命令** → 重复步骤 1-5
7. **结果**：电机在"尝试控制→失败→退出"之间快速循环
8. **表现**：电机震动/抖动

## 🔧 解决方案

### 立即修复步骤

#### 步骤1：清除错误
```python
import odrive
odrv0 = odrive.find_any()

# 清除所有错误
odrv0.clear_errors()
print(f"轴错误: 0x{odrv0.axis0.error:08X}")  # 应该显示 0x00000000
```

#### 步骤2：增加电流限制
```python
# 当前电流限制可能不够，增加到 20A
odrv0.axis0.motor.config.current_lim = 20.0
odrv0.axis0.controller.config.vel_limit = 10.0

# 保存配置
odrv0.save_configuration()
odrv0.reboot()
```

#### 步骤3：检查电源电压
```python
# 重新连接后检查电压
odrv0 = odrive.find_any()
print(f"总线电压: {odrv0.vbus_voltage:.2f} V")

# 电压应该 ≥ 12V，建议 ≥ 24V
if odrv0.vbus_voltage < 12.0:
    print("⚠️ 电压过低！请检查电源")
```

#### 步骤4：重新测试
```python
# 清除错误
odrv0.clear_errors()

# 设置合理的限制
odrv0.axis0.controller.config.vel_limit = 10.0
odrv0.axis0.motor.config.current_lim = 20.0

# 进入闭环
odrv0.axis0.requested_state = AXIS_STATE_CLOSED_LOOP_CONTROL

# 等待2秒
import time
time.sleep(2)

# 检查状态
print(f"轴状态: {odrv0.axis0.current_state}")
print(f"轴错误: 0x{odrv0.axis0.error:08X}")

# 如果没有错误，发送低速命令测试
if odrv0.axis0.error == 0:
    odrv0.axis0.controller.input_vel = 0.5  # 0.5 圈/秒
    time.sleep(5)
    
    # 停止
    odrv0.axis0.controller.input_vel = 0.0
    time.sleep(1)
    odrv0.axis0.requested_state = AXIS_STATE_IDLE
```

## 🔍 其他可能需要检查的问题

### 1. 电源问题
- **检查电源电压**：应该 ≥ 24V
- **检查电源容量**：应该能提供至少 10A 持续电流
- **检查电源线**：确保连接牢固，线径足够粗

### 2. 编码器问题
虽然编码器显示"已准备好"，但如果震动持续：
- 检查编码器连接是否牢固
- 检查编码器线缆是否有干扰
- 确认 CPR 配置正确（当前 16384）

### 3. 机械问题
- 检查电机轴是否有负载
- 检查机械连接是否牢固
- 检查是否有机械卡滞

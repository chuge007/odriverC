# zh() 函数修复说明

## 问题
`zh()` 函数已修改为直接使用 UTF-8 字符串，但所有调用处仍传入十六进制编码的字符串。

## 解决方案
有两种方法：

### 方法1：恢复 zh() 函数使用十六进制解码（推荐）
```cpp
QString zh(const char *utf8Hex)
{
    return QString::fromUtf8(QByteArray::fromHex(utf8Hex));
}
```

### 方法2：替换所有 zh() 调用为直接中文字符串
这需要大量修改，不推荐。

# ODrive 高级参数工具使用说明

## 📋 功能概述

这个Python工具提供了完整的ODrive高级参数读取和设置功能，包括：

### ✅ 核心监控参数（5个必看）
1. **vbus_voltage** - 电源电压
2. **pos_estimate** - 位置估算
3. **vel_estimate** - 速度估算
4. **Iq_setpoint** - Iq设定值
5. **Iq_measured** - Iq测量值

### ✅ PID控制参数（5个必调）
1. **pos_gain** - 位置增益
2. **vel_gain** - 速度增益
3. **vel_integrator_gain** - 速度积分增益
4. **current_lim** - 电流限制
5. **vel_limit** - 速度限制

### ✅ 扩展参数
- 编码器参数（CPR、模式、计数器）
- 电机参数（极对数、力矩常数）
- 轨迹控制参数（速度、加速度、减速度限制）

---

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install odrive
```

### 2. 连接ODrive

确保ODrive通过USB连接到电脑。

### 3. 运行工具

```bash
# 使用axis0（默认）
python advanced_params_tool.py

# 使用axis1
python advanced_params_tool.py 1
```

## 🔧 常见使用场景

### 场景1：检查电机状态
```
1. 选择 "1" - 读取核心监控参数
2. 查看电源电压是否正常（应该在20-30V之间）
3. 查看Iq电流是否合理
```

### 场景2：调整PID参数
```
1. 选择 "2" - 读取当前PID参数
2. 选择 "7" - 设置新的PID参数
3. 选择 "9" - 实时监控效果
4. 如果效果好，在odrivetool中运行 odrv0.save_configuration()
```

### 场景3：诊断问题
```
1. 选择 "6" - 读取所有参数
2. 检查各项参数是否正常
3. 特别关注：
   - 电源电压是否稳定
   - 编码器计数是否正常
   - 电流是否超限
```

## ⚠️ 注意事项

### 1. 参数保存
- 通过工具设置的参数**不会自动保存**到flash
- 如需永久保存，必须在odrivetool中运行：
  ```python
  odrv0.save_configuration()
  odrv0.reboot()
  ```

### 2. 安全提示
- 修改PID参数前，确保电机处于安全状态
- 建议先在小范围内测试参数效果
- 电流限制不要设置过大，避免损坏电机

### 3. 推荐参数值

#### 标准电机（如KILOCODE）
```
pos_gain = 8.0
vel_gain = 0.05
vel_integrator_gain = 0.2
current_lim = 20.0 A
vel_limit = 10.0 圈/秒
```

#### 大功率电机
```
pos_gain = 10.0
vel_gain = 0.08
vel_integrator_gain = 0.3
current_lim = 30.0 A
vel_limit = 15.0 圈/秒
```

## 🐛 故障排除

### 问题1：找不到ODrive
```
❌ 错误: Could not find any ODrive
```
**解决方案**：
1. 检查USB连接
2. 确认ODrive已上电
3. 尝试重新插拔USB

### 问题2：读取参数失败
```
❌ 读取监控参数失败: ...
```
**解决方案**：
1. 确认ODrive固件版本兼容
2. 检查轴号是否正确（axis0或axis1）
3. 尝试重启ODrive

### 问题3：设置参数无效
**解决方案**：
1. 确认参数值在合理范围内
2. 检查电机是否处于IDLE状态
3. 某些参数需要在特定状态下才能修改

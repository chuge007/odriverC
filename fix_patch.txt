问题2：电机不动
原因：代码尝试同时控制多个节点，但实际只有一个接口
位置：mainwindow.cpp 第3450行左右的 connectJogButton lambda

当前错误代码：
```cpp
// 获取所有配置的车轮节点ID
QList<quint8> nodeIds;
for (int i = 0; i < qMin(4, m_boardConfigRows.size()); ++i) {
    const quint8 nodeId = turnAxis ? boardTurnNodeId(i) : boardDriveNodeId(i);
    if (!nodeIds.contains(nodeId)) {
        nodeIds.append(nodeId);
    }
}
// 同时向所有节点发送速度命令
for (const quint8 nodeId : nodeIds) {
    commandAxisVelocity(boardIndex, nodeId, ...);
}
```

修复为：
```cpp
// 只控制当前板子的对应节点
const quint8 nodeId = turnAxis ? boardTurnNodeId(boardIndex) : boardDriveNodeId(boardIndex);
commandAxisVelocity(boardIndex, nodeId, direction * qMax(0.1, m_jogSpeedSpin->value()), turnAxis);
```

1. 左侧添加所有控制按钮（垂直排列）
2. 右侧显示完整的状态监控信息

```
[轴状态与保护] - 横向按钮组
[控制模式] [位置控制] [速度控制] [力矩控制] [限制] | [状态监控]
```

## 实现步骤

### 1. 修改 buildAdvancedContent 函数

将当前的布局：
```cpp
QHBoxLayout *middleLayout = new QHBoxLayout;
QVBoxLayout *leftColumn = new QVBoxLayout;
// 左侧：控制模式、位置、速度、力矩、限制
QGroupBox *statusGroup = new QGroupBox(zh("状态监控"));
// 右侧：状态监控
middleLayout->addLayout(leftColumn, 1);
middleLayout->addWidget(statusGroup, 1);
```

改为更清晰的左右布局，左侧所有控制按钮垂直排列，右侧完整状态信息。

### 2. 状态监控区域添加的完整信息

### 3. 代码修改位置

在 `mainwindow.cpp` 的 `buildAdvancedContent` 函数中：

```cpp
void MainWindow::buildAdvancedContent(QVBoxLayout *mainLayout, QWidget *parent)
{
    // 1. 轴状态与保护按钮组（保持不变）
    QGroupBox *axisStateGroup = ...;
    
    // 2. 创建左右布局
    QHBoxLayout *middleLayout = new QHBoxLayout;
    middleLayout->setSpacing(12);
    
    // 3. 左侧控制区域（垂直布局）
    QVBoxLayout *leftColumn = new QVBoxLayout;
    leftColumn->setSpacing(12);
    
    // 添加所有控制组件
    leftColumn->addWidget(modeGroup);        // 控制模式
    leftColumn->addWidget(positionGroup);    // 位置控制
    leftColumn->addWidget(velocityGroup);    // 速度控制
    leftColumn->addWidget(torqueGroup);      // 力矩控制
    leftColumn->addWidget(limitGroup);       // 运行限制
    leftColumn->addStretch(1);               // 底部弹性空间
    
    // 4. 右侧状态监控区域（已有完整信息）
    QGroupBox *statusGroup = new QGroupBox(zh("状态监控（完整信息）"));
    QGridLayout *statusLayout = new QGridLayout(statusGroup);
    // ... 现有的所有状态标签 ...
    
    // 5. 组合布局
    middleLayout->addLayout(leftColumn, 1);   // 左侧占1份
    middleLayout->addWidget(statusGroup, 2);  // 右侧占2份，更宽
    
    mainLayout->addWidget(axisStateGroup);
    mainLayout->addLayout(middleLayout, 1);
}
```

## 总结

当前代码已经包含了所有需要的状态信息，只需要：
1. 调整布局比例，让状态监控区域更宽（从1:1改为1:2）
2. 确保所有控制按钮在左侧垂直排列
3. 状态监控标题改为"状态监控（完整信息）"以强调其完整性

这样用户就能在高级页面看到：
- 左侧：所有控制功能（模式、位置、速度、力矩、限制）
- 右侧：完整的电机状态信息（17项详细状态）
